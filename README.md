# my-machine

### Run my-machine

##### Pre-requisite
Docker

##### Steps

Method 1 : 

`docker run -it auenkr/my-machine`

Method 2 :

1. `git clone https://github.com/AuenKr/my-machine.git`
2. `cd my-machine`
3. `docker build -t my-machine .`
4. `docker run -it my-machine`
