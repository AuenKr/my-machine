# my-machine

### Run my-machine

##### Pre-requisite
Docker

##### Steps

1. `git clone https://github.com/AuenKr/my-machine.git`
2. `cd my-machine`
3. `docker build -t my-machine .`
4. `docker run -it my-machine`

### Installed tools

1. Git
2. Docker
3. Node Version Manager (nvm)
4. Nodejs v20.16.0
5. Bun
6. java-sdk, cpp
7. Neovim (lazyvim)
8. Firewall manager (ufw)
9. Neofetch, htop
10. ffmpeg
